# [SEDEMAC Corporate Website](https://www.sedemac.com/)

The idea of having a corporate web-page and a web-application built around it to digitize SEDEMAC's web-presence make it vibrant, dynamic and gather data driven insights. This web-app will be used to gather visitor e-mails in exchange of product information downloads, improve careers pages, give accurate and current impression to an un-initiated visitor on work at SEDEMAC. 

## Built with

- React JS
- MUI
- Typescript
- Figma & Sketch
- 3P Integrations: Google Analytics, Notion, Confluence.

## Structure

The folder and other strucuture is as follows
- components - these are the most fundamental UI building blocks
- layouts - these use components to give structure to all pages
- views - these are the actual renderings of the pages
- svg - graphics
- theme - ui-customizations

- Install dependencies: `yarn install`
- Start the server: `yarn run start`
- Build on production: `yarn run build`
